# SCT_WD_2
Task 2 – Stopwatch web application with start, stop, and reset functionality built using HTML, CSS, and JavaScript.
Task 2 – Stopwatch Web Application

 *About the Task* 
This task involves creating a stopwatch web application that allows users to track time accurately. The application includes basic stopwatch controls such as start, stop, and reset. JavaScript is used to manage the timing logic and update the display in real time, while HTML and CSS are used to structure and style the user interface. This task helps in understanding JavaScript event handling and DOM manipulation.

 *Technologies Used* 
- HTML5 for structuring the application
- CSS3 for styling and layout
- JavaScript for stopwatch logic and interactivity

 *Features* 
- Start, stop, and reset functionality
- Displays time in hours, minutes, and seconds
- Real-time updates using JavaScript
- Simple, clean, and responsive design
